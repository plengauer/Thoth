# Demo "Deep injection into a Gradle build"

## Script
```sh

export PATH="$(pwd)":"$GRADLE_HOME"/bin:"$PATH"
export OTEL_SHELL_CONFIG_INJECT_DEEP=TRUE
. otel.sh
gradlew build
```
